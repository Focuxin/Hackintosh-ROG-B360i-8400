# Hackintosh-ROG-B360i-8400

## 配置
* CPU: i5-8400
* 主板: ROG-B360i
* 内存: 光威单条16GB
* 显卡: RX560（肥猫家）
* 网卡/蓝牙: 博通BCM94360CS2
* 显示器: AOC U2790PQU 4K 27英寸

## 主板BIOS设置
* CFGLock -> Disabled
* VT-d -> Disabled
* 大于4G地址空间解码 -> Enabled
* DVMTPre-Allocated -> 128M
* Systemtime and Alarm Source -> Legacy RTC
* SATA模式选择 -> AHCI（必须设置）
* legacyUSB支持 -> Enabled
* XHCIHand-off -> Enabled
* 快速启动 -> Disabled
* 开启CSM -> Disabled
* 安全启动状态 -> 关闭
* 操作系统类型 -> 其他操作系统
