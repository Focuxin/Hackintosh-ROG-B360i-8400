# Hackintosh-ROG-B360i-8400
## 配置
* CPU: i5-8400
* 主板: ROG-B360i
* 内存: 十铨火神DDR4-2400
* 显卡: 核显
* 网卡/蓝牙: 无

## 主板BIOS设置
* CFGLock -> Disabled
* VT-d -> Disabled
* 大于4G地址空间解码 -> Enabled
* DVMTPre-Allocated -> 128M
* Systemtime and Alarm Source -> Legacy RTC
* SATA模式选择 -> AHCI（必须设置）
* legacyUSB支持 -> Enabled
* XHCIHand-off -> Enabled
* 快速启动 -> Disabled
* 开启CSM -> Disabled
* 安全启动状态 -> 关闭
* 操作系统类型 -> 其他操作系统
